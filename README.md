🗝️ portfolio-escape-room

Welcome to Mystery Portfolio—an interactive portfolio where every archive gate hides a new chapter of my journey as a developer.
Unlike traditional portfolios, this website turns exploration into an adventure. Instead of simply scrolling through different sections, visitors must unlock archive gates by solving basic web development questions related to HTML, CSS, and JavaScript. Each correct answer opens the next archive, revealing more about me, my projects, and my skills.

📖 About This Project

Mystery Portfolio is a game-inspired web portfolio created to make showcasing my work more engaging and memorable. Visitors begin at the entrance screen and enter a 3D archive room containing multiple locked doors.

Each door represents an important section of my portfolio:

👤 About Me

💼 Projects

📩 Contact

To unlock each archive, visitors solve simple web-development puzzles. After answering correctly, the next door becomes accessible until the entire portfolio is revealed.
